SHMUP
=====

This is the source code of "SHMUP", a 3D shoot 'em up written in 2009 and
inspired by Treasure's Ikaruga. This fork keeps only the iOS version.

The main project page is at [http://fabiensanglard.net/shmup/](http://fabiensanglard.net/shmup/).

![SHMUP Splash](/engine/iOS/Default.png)

Technical side
==============

It is written in ANSI C with an iOS wrapper. Rendering is done with OpenGL ES
1.1, with an experimental OpenGL ES 2.0 path using dEngine.
